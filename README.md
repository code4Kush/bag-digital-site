# BAG Digital | Beaufort Art Glass (v1.6.0)
**Founder:** Jaime | **Location:** Beaufort, SC
**Tech Stack:** HTML5, CSS3 (Custom Variables), JavaScript (Vanilla), GitHub Pages.

## Overview
A dual-purpose business engine. 
1. **BAG Digital:** Systems architecture, AI automation, and web development.
2. **Beaufort Art Glass:** High-end Borosilicate (Boro) glass art (Pendants & Marbles).

## Architecture
- **Chameleon Themes:** Switch between `agency` (Cyber), `luxe` (Gallery), and `industrial` (Studio) via the UI.
- **Headless Commerce:** Uses Stripe Payment Links instead of expensive Shopify subscriptions.
- **Lead Capture:** Integrated form for Trello/Google Sheets automation.

## Deployment
- Hosted on GitHub Pages.
- Images served via Google Drive Direct Links.
- Inventory managed via `content.json`.
